# Mémo pour l’équipe

L’objectif du jeu est de créer un **RPG** dans lequel le joueur incarne un **héros canard**, venu pour **sauver le monde**.
Afin d’affronter le grand méchant, il devra éliminer les monstres sur son chemin.

**CoinCoinQuest** proposera une aventure captivante où le joueur devra **s’améliorer**, utiliser sa **curiosité**, et faire preuve de **courage** pour vaincre le mal.
