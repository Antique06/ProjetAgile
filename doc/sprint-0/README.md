## Rétrospective

### Sur quoi avons-nous buté ?

* Lors de ce premier sprint, nous avons bien planifié le projet. Nous disposons d’une base solide et d’une vision claire. La communication au sein de l’équipe semble également satisfaisante.
* Toutefois, nous avons perdu pas mal de temps à nous accorder sur le concept du jeu. Malgré cela, le résultat final reste très satisfaisant et motivant.

### PDCA (Plan, Do, Check, Act)

Notre objectif est de créer un jeu agréable à jouer, avec une petite histoire et surtout sans bugs.

Nous allons donc :

1. Mettre un point d’honneur sur les **tests** ;
2. Favoriser une **communication fluide** au sein de l’équipe ;
3. **Rester soudés**, malgré l’intensité du projet.

# Mémo pour l’équipe

L’objectif du jeu est de créer un **RPG** dans lequel le joueur incarne un **héros canard**, venu pour **sauver le monde**.
Afin d’affronter le grand méchant, il devra éliminer les monstres sur son chemin.

**CoinCoinQuest** proposera une aventure captivante où le joueur devra **s’améliorer**, utiliser sa **curiosité**, et faire preuve de **courage** pour vaincre le mal.
